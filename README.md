# React Users & Posts Dashboard

A simple dashboard built with **React + Vite + TypeScript** that displays users and their related posts.  
Data is fetched from the public **JSONPlaceholder API**.

---

## âœ¨ Features

- Display users in a table
- Search and sort users
- Select a user to view related posts
- Pagination support
  - Users: 5 users per page
  - Posts: 4 posts per page
- Global state management with **Zustand**
- Data fetching and caching with **TanStack React Query**
- API requests handled via **Axios**
- **Dark / Light mode** support
- Fully responsive UI using **Tailwind CSS**
